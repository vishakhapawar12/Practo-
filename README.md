1. OBJECTIVES 🎯

The main objective of this project is to:

1.Analyze a healthcare platform (Practo) and understand how it works end-to-end
2.Design a scalable database schema for managing patients, doctors, appointments, etc.
3.Solve real-world healthcare problems like accessibility, record management, and engagement.
4.Propose revenue growth strategies (target: ≥25% profit increase).
5.Perform guesstimates & data analysis (retention, revenue, cost savings).
